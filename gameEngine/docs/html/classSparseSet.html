Detailed Description

template<typename Entity, typename Component>
class SparseSet< Entity, Component >

Efficient associative container mapping entities to components.

The SparseSet is a core ECS data structure used for storing components. It provides O(1) insertion, lookup, and removal of components by entity ID.

Internally, it maintains three parallel arrays:

  • sparse: maps entity IDs to indices in the dense array
  • dense: stores the active entity IDs (contiguously)
  • data: stores the components associated with entities

This layout ensures tight memory packing and cache-friendly iteration.

Template Parameters
EntityThe entity identifier type (e.g. uint32_t).
ComponentThe component type to store (e.g. Transform, RigidBody).
