Detailed Description

Handles creation and destruction of entities within the ECS framework.

The EntityManager is responsible for generating unique entity IDs, reusing destroyed entity IDs for efficiency, and keeping track of how many entities are currently alive.

Entities are represented by simple unsigned integers (uint32_t).
