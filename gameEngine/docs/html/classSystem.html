Detailed Description

template<typename Derived>
class System< Derived >

CRTP (Curiously Recurring Template Pattern) base class for ECS systems.

This template class provides a convenient base for defining game systems. Derived classes should implement:

void onUpdate(Registry& registry, float dt);

class MovementSystem : public System<MovementSystem> {

MovementSystem() {
